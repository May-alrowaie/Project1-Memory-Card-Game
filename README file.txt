Memory Number Game:
Overview:
This is a Memory Number Game built using HTML, CSS and JavaScript. The game challenges the player to match pairs of numbers by clicking on hidden cards. It's a fun way to enhance memory and concentration skills.

Features:
Interactive card-matching gameplay.
Score tracking. 
Time-based challenge .

Technologies Used:
HTML for structuring the web page.
CSS for styling the game.
JavaScript for game logic and interactivity.

How to Play:
1-Click on a card to flip it over.
2-Try to find its matching number by flipping another one.
3-If the cards match, they will remain flipped. If not, they will flip back over.
4-The goal is to match all pairs of cards in the least amount of moves or time.

Features:
Simple number-based matching game from 1-10.
Works in any modern web browser.
No additional setup or external resources required.
Fully built with HTML, JavaScript and CSS.
